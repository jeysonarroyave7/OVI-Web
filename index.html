INTRODUCCIÓN AL DESARROLLO DE SOFTWARE


Objetivo General

Realizar un OVI a través de HTML, con el fin de conocer conceptos y manejo de herramientas para diseñar sitios web.

Objetivos Especificos

  • Dar a conocer cuáles son las etapas necesarias para el desarrollo de cualquier software.

  • Dar una breve explicación de cada una de las etapas necesarias para el desarrollo de cualquier software.

  • Reforzar la explicación de las etapas necesarias para el desarrollo de cualquier software a través de contenido multimedia.
